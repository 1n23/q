Statistics Window

 

The Statistics Window shows you information about the ressources your project eats up. If you are working on elaborate set-ups, it pays off to keep the occasional eye on the statistics to make sure you don't go overboard.

 

Tracked Statistics

 

pages

Displays the total number of pages (and the total number of operators in brackets)

 

current page

Displays the number of operators on the current page

 

in meshes

Displays the amount of memory used for 3D data

 

in bitmaps

Displays the amount of memory used for 2D data

 

Context Menu

 

The Statistics Window has a context menu that is accessible through the triangle icon in its title bar.

 

Close window

Closes the Statistics Window
