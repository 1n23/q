Parameter Window

 

The Parameter Window displays the properties of the currently selected operator (note that this does not need to be the operator currently shown in the View Window).

 

Operator Menu

 

Independently of the exact type of operator selected, the following controls, found at the top of the Parameter Window can be set for any operator:

 

Name field

You can give any operator a name, which allows it to be loaded into other parts of the document. You can alternatively use a Store Operator to name operators.

 

Redo

Redo re-applies changes after you undid them.

 

Undo

Undoes the changes of an edit session.

 

Bypass

Temporarily turns the currently selected operator into a Nop Operator.

 

Hide

Temporarily disables the currently selected operator completely.

 

Context Menu

 

The Operator Page Window has a context menu that is accessible through the triangle icon in its title bar.

 

Help

Opens this page.

 

Help on Operator

(only available if you are looking at an operator) Opens the Help page for the current operator.
