Editor Settings

 

The Editor Settings allow you to set some parameters that effect the way .werkkzeug3 texture edition looks and behaves.

 

Font Size

(small/large): Toggles the GUI font size.

 

Splashscreen

(show/hide): Toggles whether the splash screen is displayed on startup.

 

Scrollbars

(show/hide): Toggles scrollbars on all views.

 

Help system

(Internet/Local): Defines whether to use the local or online documentation.

 

Autosave Interval (min)

Determines how often your project is autosaved.

 

Texture Cache (MBytes)

Adjusts the memory cap for texture data.

 

Mesh Cache (MBytes)

Adjusts the memory cap for 3D mesh data.
