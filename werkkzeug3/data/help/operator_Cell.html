Cell Operator

 

The Cell Operator is a leaf. It creates an irregular, stained-glass-like texture.

 

Inputs

 

None.

 

Parameters

 

Width

Sets the texture width.

 

Height

Sets the texture height.

 

lock

Locks the texture resolution to 100%, overriding the the setting in the Edit menu.

 

Color 0

Defines the cell fill color.

 

Color 1

Defines the cell edge color.

 

Max

Controls the maximum amount of cells.

 

Min Distance

Controls the minimum cell size.

 

Seed

A unique identifier for the pseudorandom pattern.

 

Amplify

Controls glow size.

 

Gamma

Controls glow falloff curve.

 

Aspect

Controls general cell shape.

 

Flags

outer/inner: Glowing edges (outer) / Glowing cell centers (inner).

cell-color(on/off): Toggles cell-color mode.

invert(on/off): Swaps fill color(s) and edge color(s).

 

Cell Color Mode

 

In Cell Color Mode, Color 2 will be used for the fill of the cells, while the cell stroke colors will be taken from a gradient between Colors 0 and 1.

 

Color 2

Defines the fill color for Cell Color Mode.

 

Percentage

Controls the amount of cells that are not filled at all.
